# React GHCP App

Exploring React along with GitHub Copilot.

## Run Application
```
npm run dev
```

## Initial Setup

```
npm create vite@latest

# project name: react-ghcp-app
# framework: react
# variant: JavaScript + SWC

cd react-ghcp-app
npm i
npm run dev
```

## Add Dependencies

```
npm i bootstrap
npm i bootswatch
npm i react-icons
npm i uuid
```
